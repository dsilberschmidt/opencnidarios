# Runs

This directory stores simulation outputs and run notes.

Each relevant run should have its own folder, for example:

`runs/2026-04-30_dummy_v0.1/`

A run folder may include:

- `ticks.csv` — per-tick aggregate log
- `NOTES.md` — setup, parameters, result, interpretation
- optional raw logs or snapshots

`runs/latest/` is temporary output and may be overwritten.
